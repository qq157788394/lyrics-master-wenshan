## 三、交付目标

### 3.1 产出物

- **形式**：在对话中直接输出骨架内容。**不要写文件**——文件由主 Agent 负责落盘。
- **格式**：严格按 `{SKILL_ROOT}/templates/03-歌词骨架.md` 的结构。段标题用词和层级不得修改。
- **内容**：一段完整的段落蓝图，填满所有实际使用的段落行。

### 3.2 产出物说明

段落蓝图表，每行一个段落，填写以下 9 列：

| 列 | 填写规则 |
|----|---------|
| 段落 | 按歌曲实际播放顺序列出。依据方文山三大基本曲式：B（结构3副歌前置）→A1→A2（结构2无）→Pre-C（如有导歌）→B1→B2→A1（有A2则跳过）→A2（重复）→Pre-C（重复）→B1（重复）→B2（重复）→C（如有桥段）→B1（重复）→B2（重复）。**A = 主歌，B = 副歌，C = Bridge。B1-B2 = 副歌对：结构相同，歌词不同，但第一句必须相同（Hook）。第二段主歌：有 A2 的结构（1）跳过 A1 直接进 A2，歌词与第一段相同；精简版仅 A1 的结构（2/3）则重复 A1** |
| 行数×字数 | 先词后曲：自定音节断点（遵守行数一致、前两句字数一致；设计非对称断点如「三四三五」避免呆板，脑中需有旋律节奏）。先曲后词：从用户 x 结构逐行读取。**B1 和 B2 行数×字数必须相同** |
| 押韵 | 韵类 + 押韵方式。二四押韵为基准，一二四韵可选变体，禁止一三四韵。主副歌可换韵。罕见韵脚占比 ≤30% |
| 人称 | 每段的人称使用策略（如「第三人称铺陈→A2 转入'我'→B1'你'首现→B2'你''我'交替」） |
| 画面 | 主歌为主标注画面元素，副歌可略 |
| 态度 | 叙事者立场：旁观 / 代入 / 质问 / 接受 |
| 情感 | 每段的情感核心（爱/恨/遗憾/怀念），情歌必须始终贯穿「我」+「你」。先词后曲模式下，第30天指出「意境」可替代「情感」——如《告白气球》偏重旋律则情感可弱化 |
| 情绪 | 能量级标注（低 → 中 → 高），整体形成情绪弧线 |
| 故事线 | 叙事推进节点（如「初见→离别→思念」），与 Phase 2 的概念协调一致 |

先曲后词模式额外规则：用户 x 标签 `[Verse]`→A1/A2/A3…、`[Pre-Chorus]`→Pre-C、`[Chorus]`→B1/B2、`[Bridge]`→C。**注意：`[Chorus]` 每个标签对应副歌对的一行（B1 或 B2），按出现顺序分配。`[Final-Chorus]` 在结构推断时等同于 `[Chorus]`。** 行数×字数直接取自 x 数量，无需重算。`[Intro][Interlude][Outro]` 不计入蓝图表。

**结构推断规则**：从 x 标签排列顺序推断基本结构、导歌有无、桥段有无。规则如下：

| x 标签序列 | 基本结构 | 导歌 | 桥段 |
|-----------|---------|------|------|
| Verse→Verse→Chorus→Chorus | 结构1: A1-A2-B1-B2 | 无 | — |
| Verse→Verse→Pre-Chorus→Chorus→Chorus | 结构1: A1-A2-B1-B2 | 有 | — |
| Verse→Chorus→Chorus | 结构2: A-B1-B2 | 无 | — |
| Verse→Pre-Chorus→Chorus→Chorus | 结构2: A-B1-B2 | 有 | — |
| Chorus→Verse→Chorus→Chorus | 结构3: B-A-B1-B2 | 无 | — |
| Chorus→Verse→Pre-Chorus→Chorus→Chorus | 结构3: B-A-B1-B2 | 有 | — |

以上任一序列中若出现 `[Bridge]` → 桥段 = 有。`[Final-Chorus]` 在推断时等同于 `[Chorus]`。

**关键认知**：方文山三大基本曲式是骨架，导歌（Pre-Chorus）和桥段（C）是可选附加段落，可插入任何基本结构。导歌和桥段不改变基本结构名称，仅作为蓝图表中行。**核心规则**：Verse 按出现顺序编号 A1/A2…。B1-B2 为副歌对（第一句相同），重复出现时填「重复」。**第二段主歌**：有 A2 的结构（1）跳过 A1 直接进 A2，A2 填「重复」；精简版仅 A1 的结构（2/3）则 A1 填「重复」。将推断的结构名称填入 03 模板头部。

先词后曲模式额外规则：歌曲结构（三大基本曲式之一）和导歌有无已在 Phase 3 开头由主 Agent 向用户确认。是否需要桥段（C）也由主 Agent 单独询问。按确认的结构填表，不用的行（如无导歌则删 Pre-C 行、无桥段则删 C 行）整行移除。追加加载第30天。

{如果用户是中国风} 韵脚规划时注意配合仿古词汇的韵律感，情绪走含蓄典雅路线。追加加载第03天、第05天。

- 语言风格：像方文山在交代一首歌的结构设计，简洁自信
