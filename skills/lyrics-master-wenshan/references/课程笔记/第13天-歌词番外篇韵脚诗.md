# 第13天-歌词番外篇韵脚诗



## 一、知识小结



| 主题 | 核心内容 | 诗歌特点 | 教学重点 |
|-|-|-|-|
| 诗与歌词的区别 | 歌词可入乐，诗的结构形式不适合直接入乐 | 诗的文字洁净度高，有独特排列组合 | 创作养分吸收方式 |
| 素颜韵脚诗 | 无标点符号、无数字、纯净汉字表达，行进间押韵脚，有节奏感 | 方文山独创诗体 | 诗歌创作意义：个人文字日记，反映心态心情，可独立成完整作品 |
| 经典诗歌赏析 | 韩瀚《重量》、北岛《波兰来客》 | 简短有力，直击心灵 | 文字力量示范 |
| 席慕蓉诗歌 | 《一棵开花的树》 | 清新风格，女性视角，描写爱情 | 文字美感培养 |
| 郑愁予诗歌 | 《错误》（被罗大佑谱曲） | 画面感强，意象丰富 | 诗与歌的转换 |
| 夏宇诗歌 | 《爱情》 | 短小精悍，巧妙转折，意味深长 | 诗歌结构创新 |
| 顾城诗歌 | 哲学思考 | 表达简洁深邃 | 思想深度挖掘 |
| 韵脚诗创作 | 《只为》《有谁看见我的日记》 | 情感细腻，隐喻丰富 | 个人风格建立 |
| 示范诗解析 | 《17岁那一年》 | 校园回忆，听觉意象，视觉化处理 | 诗歌技巧运用 |
