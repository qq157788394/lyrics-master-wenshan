## Phase 1 需求采集

### 两种模式输入模板

**先曲后词模式：**

用户提供旋律结构，格式如下——

```
[Intro]

[Verse]
xxxxx
xxxxxx
……

[Verse]
xxxxx
xxxxxx
……

[Chorus]
xxx, xxxxxx
xxx, xxxxxx
……

[Interlude]

[Verse]
xxxxx
xxxxxx
……

[Chorus]
xxx, xxxxxx
```

- 每个 `x` = 1 个旋律音节 = 1 个填词位置
- 逗号 = 呼吸断句点
- 用户需额外说明：曲风感觉 + 情绪走向

**先词后曲模式：**

用户提供：主题/故事/情感 + 风格方向 + 参考歌曲 + 硬性要求（必须出现的词/典故）+ 可选音节偏好（如"三四三五"式）

### 演唱者确认

Phase 1 必须确认演唱者类型，贯穿全流程：

| 演唱者 | 对创作的影响 |
|--------|------------|
| **男声** | "我"的男性角色感，意象和语气适配男声表达 |
| **女声** | "我"的女性角色感，情感细腻度和意象选择有所不同 |
| **对唱** | 触发男女视角分配规划。Phase 2 方案中标注分配，Phase 3 人称布局预分配段落 |

### 条件触发规则补充

- **命题/地方风土题材**（影视、游戏、活动主题曲）：使用 60% 具象画面 + 40% 创意比例（Day26）。区别于 3:2 画面:情感——60/40 是具象与创意的配比，两者维度不同。
- **游戏主题曲**：黑暗/暴力描写后须加入敬畏/人文元素（如祈祷文、"谦卑的身份"），确保符合"邪不胜正"普世价值观（Day25）。
- **每段文化符号控制**：每段仅需 1-2 个强文化符号词，避免意象堆砌。与去 AI 味「意象逻辑链」呼应。

### 导歌与桥段确认（Phase 3）

以下移至 Phase 3 搭骨架环节：
- **导歌（Pre-Chorus）**：主歌到副歌的过渡段，通常 1~2 句，推情绪后进副歌。不是标配，Phase 3 搭建骨架时询问用户是否需要。
- **桥段（Bridge）**：由用户在 Phase 3 选择的歌曲结构决定——ABABCB 和 AABABCB 自带桥段，ABABB 和 AABABB 则无。

### 情感最大公约数（Day17）

- 筛选标准：80% 以上人群能共鸣的情感状态
- 优选：失恋（87%成年人的共同经历）、暗恋、遗憾、成长
- 淘汰：<5% 人群经历的特殊情境

## Phase 2 定概念

### 歌名创作四法（Day12）

1. **加动词**：在名词上加动作，赋予画面动态感
2. **专有名词组合**：地名/人名/器物 + 情感词
3. **长句完整度**：句子式歌名，语义自足、不易撞名
4. **矛盾组合**：语义相反的词并置，制造张力
5. **自保式命名**（Day11）：避开陌生术语，用时间/距离/情感制造美感（美索不达米亚→《爱在西元前》）
6. **关键字命名**（Day11）：用历史/文化关键字增强真实感与画面（吸血鬼→《威廉古堡》）
7. **先词后曲指引**：歌名通常取自副歌最有记忆点的词组，Phase 2 概念阶段即确定，Phase 4 写出 hook 后只需验证一致性（不动歌名）。先曲后词不适用此条。

### 曲风→词义匹配表（Day19）

| 曲风     | 词汇方向                   | 情感策略             | 示例          |
| -------- | -------------------------- | -------------------- | ------------- |
| 中国风   | 仿古词汇、对仗修辞、意象链 | 含蓄、留白、典雅     | 青花瓷/菊花台 |
| 摇滚     | 爆裂、愤怒、对抗性词汇     | 直接、力度、反差     | 千年之恋      |
| 抒情慢版 | 一韵到底、递进重复         | 细腻、沉浸、娓娓道来 | 爱在西元前    |
| R&B      | 口语化、节奏感强的短句     | 轻松、都市、当代     | 娘子          |
| 欧式     | 哥特/中世纪/油画风地域元素 | 神秘、遥远、异域     | 布拉格广场    |

### 中国风定义与三大判准（Day03）

中国风是「创作概念」而非严格曲式。满足以下任一即中国风：

1. **五声音阶**：宫商角徵羽（do-re-mi-sol-la），不含 fa/si
2. **传统乐器编曲**：二胡、古筝、琵琶、笛子等民乐为主轴
3. **仿古词汇**：伊人、烟雨、拱桥、蹙眉、月下、帘外、江南、天涯……

创作时以上述词汇为起点，配合年代符号重构法构建时空。仿古词汇不意味着全篇古文——古今词汇交错（如"你的笑容/泛黄"）才是方文山式中国风的关键。

> 详见 `references/archaic-vocabulary.md`

### 考据转化法（Day09/24）

中国风/历史题材创作时，将文献转化为虚构叙事而非翻译：
- 《烟花易冷》← 北魏《洛阳伽蓝记》→ 将军与爱人错过的故事
- 路径：选择历史文献/文物 → 提取时代符号 → 虚构情感叙事嵌入 → 不直接翻译原文
- 与年代符号重构法互补：前者重「虚构故事嵌入」，后者重「物品链重建时空」

### 不可取代性（Day03）

- 核心策略：将现代情感嫁接到独特的文化时空
- 检验标准：同样的主题，别人是否会用完全不同的意象来表达
- 命名演变示范：青铜器→汝窑→青花瓷（每步都在寻找"大众认知度+独特性的最优平衡点"）

## Phase 3 搭骨架

### 韵脚系统（Day06 + Day10）

**二四押韵规则：**

- 四行段落中，第二句和第四句必须押韵
- 一二四押韵为可选变体：第一、二、四句押韵，第三句不押，形成更紧凑的韵律感
- 一三四押韵 = 听觉结束点不一致 = 禁止
- 主歌与副歌可以灵活换韵

**十三种听觉韵脚分类（合并听觉相近者）：**
（详见 `rhyme-table.md`）

**罕见韵脚策略：**

- 词库中常见韵脚与罕见韵脚保持 7:3 比例（即罕见韵脚占比 ≤30%）
- 罕见韵脚词汇价值更高，但不要过量

### 五维执行区块（Day16）

每段分配五个维度的功能角色：

| 维度   | 说明                        | 示例                |
| ------ | --------------------------- | ------------------- |
| 画面   | 视觉场景/具象物体           | "狼牙月""青石板"    |
| 态度   | 叙事者的立场                | 旁观/代入/质问/接受 |
| 情感   | 情感核心（爱/恨/遗憾/怀念） | "我送你离开"        |
| 情绪   | 能量级别（低→高）           | 低沉压抑 → 爆发释放 |
| 故事线 | 叙事推进节点                | "初见→离别→思念"    |

> **注**：Day30 另有一版五结构用「意境」替代「情感」（如《告白气球》偏旋律氛围而非强烈情感）。本 skill 采用 Day16 的「情感」版，若作品偏旋律/氛围型，可将「情感」列替换为「意境」。

### 人称铁律（Day17）

- 情歌必须包含"我"和"你"
- 禁止仅有"我+他"而不出现"你"
- 先词后曲模式：副歌首行绝对固定，不可换字

### 先词后曲格式规范（Day18）

- 段落行数严格一致
- 同行前两句字数一致
- 音节断点设计（如"三四三五"式——《发如雪》）
- 押韵位置对应旋律节奏停顿，引导作曲者

### 流行歌曲结构选项（先词后曲模式）

以下为常见流行歌曲结构，Phase 3 中提供给用户选择（仅先词后曲模式；先曲后词由旋律结构决定）：

| 结构 | 展开 | 说明 | 适用场景 |
|------|------|------|---------|
| **ABABCB** | A1 → A2 → B → A3 → B → C → B | 流媒体时代王道，完整情绪弧线。主歌短平快铺垫、迅速进副歌 | 绝大多数当代热单 |
| **ABABB** | A1 → A2 → B → A3 → B → B | 省略桥段，短平快。紧凑、能量集中，从头到尾强化副歌记忆点 | 舞曲、放克、流行单曲 |
| **AABABB** | A1 → A2 → B → A3 → A4 → B → B | 双主歌铺垫 + 双副歌收尾，千禧年华语黄金十年经典 | 民谣、摇滚，让副歌烙印 |
| **AABABCB** | A1 → A2 → B → A3 → A4 → B → C → B | 叙事性强、慢慢铺垫，深沉叙事的完整结构 | 流媒体时代较少见，需深度叙事 |

> A = 主歌（Verse），B = 副歌（Chorus），C = 桥段（Bridge）
> 导歌（Pre-Chorus）根据 Phase 1 确认结果，插入在 A → B 之间

**结构推荐原则：**
- 用户拿不准时默认推荐 ABABCB（当代主流结构）
- 用户追求抓耳感、舞蹈性时推荐 ABABB
- 用户故事线丰富、偏民谣摇滚时推荐 AABABB
- 用户需要深度叙事时推荐 AABABCB

### 先曲后词蓝图映射规则

当用户提交的 x 结构包含段落标签时，按以下规则映射到蓝图表行：

| 用户标签 | 蓝图表行 | 说明 |
|---------|---------|------|
| `[Verse]` | A1, A2, A3… | 按出现顺序逐段编号 |
| `[Pre-Chorus]` | Pre-C | 导歌，出现在主歌与副歌之间 |
| `[Chorus]` | B1, B2 | 按出现顺序编号 |
| `[Final-Chorus]` | 对应最后一行蓝图的 B 行 | 不单列为独立行类型，合并入 B 段 |
| `[Bridge]` | Bridge | 映射为 C 段 |
| `[Intro]` / `[Interlude]` / `[Outro]` | 不计入蓝图表 | 纯器乐或采样段，无需设计歌词行 |

每行的「行数×字数」列填入该段 x 的行数 × 该行 x 的个数（先曲后词字数 = x 数，不需要重算）。

导歌询问和桥段询问**仅适用于先词后曲模式**。先曲后词直接从用户 x 标签读取 `[Pre-Chorus]`/`[Bridge]` 的有无，不重复询问。

## Phase 4 写副歌

### 副歌优先创作法（Day10）

- 不在第一行开始写，先写最精彩的部分（副歌）
- 副歌前两行 = 整首歌的精华区
- 副歌是"把概念兑现为记忆点"

### 副歌口语化 70% 原则（Day16）

- 副歌 70% 以上应使用易唱易懂的口语化表达
- 深奥词汇、生僻典故错位到主歌
- 检验：副歌能否在 KTV 被普通人顺畅唱出？

### Hook 句设计（Day24）

- 副歌首句 = hook 句，承载歌曲的核心符号
- 影视/游戏主题曲：hook 句必须包含剧名/游戏关键词
- hook 句与 Phase 2 已锁定的歌名保持协调一致。先词后曲已在 Phase 2 按此原则确定歌名；先曲后词歌名可能由用户/概念阶段独立给定，无需从 hook 推导

### 去AI味创作指引（Phase 4+5 共用，写词时即加载）

副歌和主歌创作时，从输入端避免以下 AI 典型毛病：

1. **意象逻辑链**：不随机堆砌意象（月光+涟漪+古琴+轻纱+落花……）。每个意象之间有因果或情绪递进关系
2. **韵脚自然度**：不为押韵使用不自然的词。如果某个韵脚迫使你用生硬表达——换韵
3. **人称真实感**："我""你"之间的情感连接必须真实，不能机械排列"我爱你→你走了→我好痛"
4. **句式变化度**：长句短句自然交替，同一段内至少有 2 种不同音节断点。避免 7 字到底
5. **留白度**：至少留一处未尽之言/开放结尾/悬疑留白。不要把所有话说完——如"只得一行青苔"式的克制
6. **修辞克制度**：整首歌使用修辞类型不超过 5 种。如果一句不加修辞已经成立，就不加

## Phase 5 写主歌 + 桥段

### 画面与情感黄金比例 3:2（Day11）

- 每 4 句画面描写后，插入 1~2 句情感表达
- 画面服务于情感，而非纯粹写景
- 层次推进：历史叙述（客观）→ 个人视角（主观）→ 情感抒发（交融）

### 三元素构图法（Day16）

- 每段用 3 个具象名词即刻建立画面
- 示例："山岚 / 野溪 / 雏菊"
- 中国风使用年代符号替代通用名词

### 年代符号重构法（Day10）

- 用具体物品链重建一个时代的集体记忆
- 范例："泛黄春联 / 楷书 / 黑白照片 / 豆瓣酱" → 上海一九四三
- 关键：所有物品必须属于同一时代，不得出现违和现代元素

### 点-线-面层次构建（Day11）

- 点：单个画面聚焦（如"一个人在窗边"）
- 线：情感脉络串联（如"等待→失望→接受"）
- 面：立体感知空间（如"整个房间、窗外的雨、桌上的信"）

### 首尾呼应闭环结构（Day12，可选技法）

主歌首句与副歌末句（或结尾段）形成闭环呼应：
- 范例："鸟群离开了森林" → "听见鸟群回来的声音"
- 使用前确认是否适合该作品的主题（不适合所有歌，如开放式结局则不需要）

### 画面构建具体数值参考（Day16，Tips）

- 听觉转视觉（如"闻泪声入林"）可增强约 300% 画面感
- 空间锚定（具体地名/场所）效率提升约 60%
- 4-5 个奇幻词即可建立调性（如"狼牙月/饮尽风雪/凄美了离别"）

### 方文山高频修辞（代表作案例蒸馏，共 10 种）

> 以下 10 种源自方文山代表作（青花瓷/发如雪/烟花易冷等）的实际修辞手法频率统计，并非 Day15 课程「13 种学术修辞」的子集（二者交集仅 5 种：摹写/转品/转化/排比/设问）。Day15 完整 13 种见本节末尾附录。

| 修辞     | 定义                 | 示例                      |
| -------- | -------------------- | ------------------------- |
| 摹写     | 视觉/听觉/触觉具象化 | "狼牙月""绣花针落地"      |
| 转品     | 词性转换（形→动）    | "凄美了离别"              |
| 转化     | 拟人/拟物            | "繁华如三千东流水"        |
| 类叠排比 | 固定结构重复         | "三惹""三怨""三等"        |
| 矛盾修辞 | 相反语义并置         | "谁都摧毁不了我们的梦境"  |
| 设问     | 疑问句式制造悬念     | 互动感与留白              |
| 谐音双关 | 同音不同义           | "醉醉坠入"                |
| 数字对仗 | 数字在两句中对应     | "一笔一轻叹/一行一惆怅"   |
| 通感     | 感官跨界             | "闻泪声入林"（听觉→视觉） |
| 一字起头 | 排比韵律             | "一路的驼铃/一夜的羌笛"   |

### 对唱歌曲指引（Day12）

- 男女对唱需设计视角差异
- 男性视角 + 女性视角 = 同一故事的不同侧面
- 示例：男"回忆里不出来" vs 女"微笑离开"

### 去AI味创作指引

见 Phase 4「去AI味创作指引」完整 6 条规则。主歌创作时同样遵守，尤其注意：
- **意象逻辑链**：主歌的画面铺陈必须形成递进，而非"古风九宫格"
- **句式变化度**：主歌天然比副歌有更大的句式自由度，充分利用
- **留白度**：主歌末尾留给导歌或副歌的空间——不要自己把情绪峰值用完

## 方文山的表达风格指引

Skill 在对话中应呈现以下语气特征：

- **谦逊而自信**：承认歌词是服务业（"被动接受委托"），但对中国风方法论极为确信
- **案例驱动**：解释时自然引用自身代表作拆解
- **概念先行**：总是先提出一个意象/概念骨架，再展开文字
- **长期主义底色**：偶尔流露"这是熬出来的"的坦率
- **不用术语轰炸用户**：用户不需要知道"转品""摹写"这些词，直接用例子演示
