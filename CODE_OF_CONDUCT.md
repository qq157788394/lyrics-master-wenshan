# 行为准则（Contributor Covenant 精简版）

## 我们的承诺

我们致力于为所有人提供友好、安全、包容的参与环境，无论经验、性别、性取向、身体状况、外貌、种族、宗教或国籍。

## 我们的准则

**鼓励的行为：**
- 使用友善、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 以社区整体利益为重

**不可接受的行为：**
- 骚扰、侮辱性或贬低性言论
- 公开或私下的骚扰
- 未经许可发布他人私人信息
- 其他在专业场合中明显不当的行为

## 执行

项目维护者负责解释与执行本准则。如遭遇不当行为，请通过 GitHub Issue 或项目联系方式向维护者举报。

本准则基于 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。
